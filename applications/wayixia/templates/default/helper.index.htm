如何使用

使用挖一下前，需要注册挖一下用户，也可以使用微薄账号登陆。


1. 安装书签采集工具

采集工具安装步骤请点下面的按钮，按照说明操作。

安装 >>


2. 采集网站图片

在安装好书签采集工具后，浏览网站时遇到不错的图片，只要点击书签栏的“挖一下”书签，会立即列举页面上的图片, 鼠标移动到你喜欢的图片上，点击"挖一下"按钮


3. 浏览图片

登录挖一下后，进入个人中心，可以查看到你创建的画集列表，点击画集进入浏览图片。系统默认会分配一个待分类的画集存储还有分有分类的图片
